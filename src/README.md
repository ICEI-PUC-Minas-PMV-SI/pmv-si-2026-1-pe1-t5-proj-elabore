# Instruções de Utilização

## Instalação do Site

O e-LABORE será implementado como um projeto estático em HTML, CSS e JavaScript. Após a criação das páginas, o sistema poderá ser executado diretamente pelo arquivo `index.html` ou por um servidor local, como a extensão Live Server do Visual Studio Code.

## Fluxo Principal

1. Abrir a landing page.
2. Acessar a tela de login/cadastro.
3. Enviar ou pular a foto.
4. Preencher detalhes pessoais.
5. Preencher mini-bio.
6. Cadastrar experiência, educação e habilidades.
7. Revisar a prévia do currículo.
8. Baixar o PDF final.
