# Projeto e-LABORE

`CURSO: Sistemas de Informação`

`DISCIPLINA: Projeto - Aplicações Web`

`SEMESTRE: 1º`

O projeto e-LABORE consiste no desenvolvimento de uma aplicação web com foco social, voltada para a criação de currículos profissionais de forma simples, gratuita e acessível. A proposta central é permitir que qualquer pessoa, especialmente aquelas em situação de vulnerabilidade ou com baixa familiaridade com tecnologia, consiga gerar um currículo estruturado por meio de um fluxo guiado, responsivo e fácil de preencher.

A aplicação contará com uma etapa de login/cadastro simples entre a landing page e o início do preenchimento do currículo. Como o projeto não prevê módulo de backend, essa etapa será tratada no front-end com persistência local quando necessário, permitindo identificar o usuário durante a navegação e preservar a continuidade da experiência. Após acessar o sistema, o usuário poderá preencher seus dados, acompanhar a prévia do currículo em tempo real e baixar o documento final em PDF.

O projeto surge como resposta a um problema real do contexto brasileiro: milhões de pessoas enfrentam dificuldades para entrar ou retornar ao mercado de trabalho não apenas por falta de experiência, mas por não conseguirem apresentar suas informações de forma adequada em um currículo. Assim, o e-LABORE busca reduzir barreiras tecnológicas e cognitivas, oferecendo uma solução intuitiva, acessível e alinhada às boas práticas de recrutamento, contribuindo diretamente para a empregabilidade e inclusão digital dessa população.
